# Constellation Plots in KNIME: An Automated Scaffold-Based Workflow for Interactive Chemical Space Visualization
## Specifications: 
The workflow was implemented using KNIME Analytics Platform (v. 5.4.2), which can be downloaded for free at https://www.knime.com/downloads. Some nodes used are part of the KNIME Base Nodes, and some others are part of specific free-use extensions such as: KNIME Statistics Nodes (Labs), RDKit Nodes Feature or  KNIME SBG Support.
The workflow can be downloaded in - .knwf - format, which is an specific type file for KNIME.
